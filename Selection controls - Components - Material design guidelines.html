选择控件

选择控件允许用户选择选项。

本指南涵盖了三种选择控制：

  • 复选框 允许从一组中选择多个选项。
  • 单选按钮 允许从一组中选择单个选项。
  • 开关 允许打开或关闭选择。

颜色

选择控件使用应用程序的重音颜色。.

主题

选择控件可用于黑暗和浅色主题。

目录

  • 复选框
  • 单选按钮
  • 开关

复选框

复选框允许用户从一组中选择多个选项。

如果列表中有多个选项，则可以使用复选框而不是开/关开关来保留空间。

如果您有一个选项，请避免使用复选框，使用开关。

亮主题

选中: 色板 500, Opacity 100%
未选中: #000000, Opacity 54%
不可用: #000000, Opacity 26%

暗主题

选中: 色板 500, Opacity 100%
未选中: #FFFFFF, Opacity 70%
不可用: #FFFFFF, Opacity 30%

单选按钮

否则，请考虑使用比显示所有选项更少的空间的下拉列表。

亮主题

选中: 色板 500, Opacity 100%
未选中: #000000, Opacity 54%
不可用: #000000, Opacity 26%

暗主题

选中: 色板 500, Opacity 100%
未选中: #FFFFFF, Opacity 70%
不可用: #FFFFFF, Opacity 30%

开关

开/关开关切换单个设置选项的状态。 应该从相应的内联标签中清楚开关控制的选项及其状态。 开关采用与单选按钮相同的视觉特性。

包含在资产中的文本“on”和“off”的开/关幻灯片切换已被弃用。 请改用这里所示的开关。

亮色

Thumb On: Swatch 500, Opacity 100%
Track On: Swatch 500, Opacity 50%

Thumb Off: Grey 50, #FAFAFA, Opacity 100%
Track Off: #000000, Opacity 38%

Thumb Disabled: Grey 400, #BDBDBD, Opacity 100%
Track Disabled: #000000, Opacity 12%

暗色

Thumb On: Swatch 200, Opacity 100%
Track On: Swatch 200, Opacity 50%

Thumb Off: Grey 400, #BDBDBD, Opacity 100%
Track Off: #FFFFFF, Opacity 30%

Thumb Disabled: Grey 800, #424242, Opacity 100%

Track Disabled: #FFFFFF, Opacity 10%

仅使用有利于手指触摸的外形的外径向反应，其中相互作用可能完全阻碍元件。 对于桌面使用鼠标，您不需要这个额外的指示。
